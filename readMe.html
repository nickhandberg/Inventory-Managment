Home Reflection

The purpose of this site will be to manage the inventory of Bob's Guitar Shop (fictional shop). The users will be able to add, remove, and update products through forms.

The 5 W's are explained below:

  1. Who: The user will be the inventory management team from Bob's Guitar Shop
  2. What: This is a inventory managment system
  3. When: The site will be incrementely completed over the course of the semester
  4. Where: The management system will be accessible online and will be using an online database
  5. Why:: The management system will allow the inventory managers access to updated information on their information at any time via the website

Database Design Images
